Wat is er verkeerd in onderstaand HTML fragment?

<.ul>
<.li>een<./.li>
<.li>twee<./.li>
<.ol>
<.li>jan<./.li>
<.li>piet<./li>
<.li>joris<./li>
<./.ol>
<.li>vier<./li>
<./.ul>

*NOTE: Om de code te kunnen zien werd er een "." in de tag geplaatst.
een "ol"-tag (genummerde lijst) kan niet worden geopend zonder dat de al-geopende "ul"-tag gesloten wordt.
De code hieronder, daarintegen, is wel plausibel.

<.ul>
<.li>een<./li>
<.li>twee<./li>
<./ul>
<.ol>
<.li>jan<./li>
<.li>piet<./li>
<.li>joris<./li>
<./ol>
<.ul>
<.li>vier<./li>
<./ul>

*NOTE: Om de code te kunnen zien werd er een "." in de tag geplaatst.
Dit resulteert in:
  • een
  • twee
  1. jan
  2. piet
  3. joris
  • vier
